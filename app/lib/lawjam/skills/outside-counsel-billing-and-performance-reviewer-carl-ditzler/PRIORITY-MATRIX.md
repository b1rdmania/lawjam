# Priority Matrix

## 1. Invoice Approval / Dispute Prep
Highest priority:
1. Rates above approved levels
2. Missing discounts / misapplied discounts
3. Prohibited expenses and surcharges
4. Unapproved timekeepers
5. Block billing / vague narratives
6. Duplicate attendance / excess conferencing
7. AI / database / application / tech tool charges inconsistent with rules
8. Budget overruns tied to questionable billing

## 2. Budget Control / Forecasting
Highest priority:
1. Spend vs budget
2. Phase overruns
3. Forecast drift
4. Staffing expansion
5. Rate increases
6. Repeated research or drafting churn
7. Cost-driver analysis

## 3. Vendor / Firm Performance Review
Highest priority:
1. Staffing efficiency
2. Budget predictability
3. OCG compliance rate
4. Narrative quality / approvability
5. Discount realization
6. Rate competitiveness
7. Matter execution efficiency
8. Relationship and panel implications

## 4. GC / Executive Review
Highest priority:
1. Overall spend trajectory
2. Major compliance issues
3. Staffing and rate mix reasonableness
4. Budget and quarter-end risk
5. Firm value / predictability indicators
6. Recommended actions

## 5. Procurement / Finance Review
Highest priority:
1. Commercial term compliance
2. Realized discounts
3. Rate governance
4. Invoice hygiene
5. Disputed-value concentration
6. Repeat issue categories

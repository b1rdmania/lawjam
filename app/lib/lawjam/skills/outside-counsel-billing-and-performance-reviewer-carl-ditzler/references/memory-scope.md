# Memory Scope

Use persistent memory narrowly for recurring preferences and administrative context only.

## Safe to Remember
- Fiscal-quarter definition
- Preferred audience and report tone
- Preferred output formats
- Panel-firm list
- Standard matter taxonomy labels
- Recurring OCG exception categories if approved by the user
- Whether public web research is generally allowed

## Do Not Persist
- Invoice-level confidential details
- Privileged matter facts
- Disputed line-item details
- Personally sensitive information, PII
- Unresolved conclusions that may change with new evidence

## Memory Rule
If a fact would be unsafe to repeat outside the current review, do not store it as persistent memory.
